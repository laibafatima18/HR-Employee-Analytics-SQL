HR Employee Analytics using SQL

📌 Project Overview

This project analyzes HR employee data using SQL to understand workforce distribution, employee attrition, age, experience, education, gender, city, payment tier, and other HR-related insights.

The project includes data understanding, data cleaning, and business analysis using SQL queries.

---

 📂 Dataset

- Dataset Name: Employee Dataset
- Total Records: 4653
- Tool Used: MySQL Workbench
